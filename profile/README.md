# EthereaLogic — Enterprise Data Trust

Open-source data quality and governance controls for Databricks Medallion Architectures. Every repo runs on Databricks Free Edition. Every claim is backed by passing tests.

## The Enterprise Data Trust Portfolio

### [DriftSentinel](https://github.com/Org-EthereaLogic/DriftSentinel) — Chapter 4
A Databricks-deployable platform that unifies intake certification, drift gating, and control benchmarking into a single governed pipeline. 355+ passing tests. Operator dashboard included.

### [Trusted Source Intake](https://github.com/Org-EthereaLogic/trusted-source-intake) — Chapter 1
Certifies every record before downstream consumption. 7 contract checks, replay detection, schema drift handling, and quarantine with explicit reasons. 56 passing tests.

### [Silent Failure Prevention](https://github.com/Org-EthereaLogic/silent-failure-prevention) — Chapter 2
Detects when business columns collapse despite healthy schema and row counts. Distribution stability scoring, 6 publication gates, and blocked Gold refresh on health score degradation. 50 passing tests.

### [Measurable Control Effectiveness](https://github.com/Org-EthereaLogic/measurable-control-effectiveness) — Chapter 3
Scores data controls against known failure scenarios with precision, recall, and ground truth. Custom approach achieved perfect recall where industry baselines missed injected drift. 37 passing tests.

### [AetheriaForge](https://github.com/Org-EthereaLogic/AetheriaForge)
Intelligent data transformation engine for Databricks — precision transforms with full audit lineage.
